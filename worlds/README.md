# ANDERGO Language Worlds

Esta carpeta separa el contenido por idioma para que ANDERGO no mantenga todos los textos dentro de un solo archivo.

Estructura recomendada:

- `worlds/english/content.js` — Mundo English.
- `worlds/spanish/content.js` — Mundo Español.
- `worlds/french/content.js` — Mundo Français.
- `worlds/italian/content.js` — Mundo Italiano.
- `worlds/german/content.js` — Mundo Deutsch.

Cada mundo puede crecer con subcarpetas futuras, por ejemplo:

```text
worlds/english/
  content.js
  a1/
  a2/
  b1/
  b2/
  c1/
  c2/
```

Esto permite ampliar lecturas, diálogos, vocabulario, gramática y ejercicios sin mezclar todos los idiomas en una sola carpeta ni cargar contenido innecesario dentro del archivo principal.
